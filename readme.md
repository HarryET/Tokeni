# Tokeni

Tiny Web Tokens For Everyone!

## Create Token
```ts

import Tokeni from "tokeni"

let token = Tokeni.encrypt("id", {}, "password");

```

## Validate Token
```ts

import Tokeni from "tokeni"

let validatedTokenData = = Tokeni.validate("token", {}, "password");

```
